1. Replace division operations like "$grid-spacing/2" with "calc($grid-spacing / 2)"

Example in _includes/_gallery.scss:
```scss
// FROM:
margin-left: -$grid-spacing/2;
margin-right: -$grid-spacing/2;
padding: $grid-spacing/2;

// TO:
margin-left: calc(-1 * $grid-spacing / 2);
margin-right: calc(-1 * $grid-spacing / 2);
padding: calc($grid-spacing / 2);
```

2. Replace transparentize() function with color.adjust()

Example in _typography.scss:
```scss
// FROM:
text-decoration-color: transparentize($color: $accent-color, $amount: 0.8);

// TO:
text-decoration-color: rgba($accent-color, 0.2);
```

Example in _buttons.scss and _component__header.scss:
```scss
// FROM:
background-color: transparentize($accent-color, 0.92);
background-color: transparentize($accent-color, 0.86);

// TO:
background-color: rgba($accent-color, 0.08);
background-color: rgba($accent-color, 0.14);
```

3. Replace darken() function with color.adjust() or a direct rgba value

Example in _buttons.scss:
```scss
// FROM:
background-color: darken($color: $accent-color, $amount: 5);

// TO:
background-color: rgba($accent-color, 0.95);
```

Important notes:
1. These changes should be made only after copying all files and testing in a separate branch
2. The rgba() function is a standard CSS function supported in all browsers and doesn't cause deprecation warnings
3. Jekyll's styling system is sensitive to syntax issues, so be careful with editing
